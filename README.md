# Simple Interest Calculator

This Python program calculates the simple interest using the formula: SI = (P * R * T) / 100

Where:
- P is the principal amount
- R is the rate of interest
- T is the time period

## Requirements

- Python 3.x

## How to Run

1. Navigate to the directory containing the file.
2. Run the following command:

   ```
   python "simple intereset.py"
   ```

3. Enter the principal amount, rate of interest, and time period when prompted.

## Example

Enter the principal amount: 1000  
Enter the rate of interest: 5  
Enter the time period: 2  
The simple interest is: 100.0
